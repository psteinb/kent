Description

This track indicates differences in genome assembly GRCh37/hg19 when compared to the next version GRCh38/hg38. Contigs used in this assembly, GRCh37/hg19, that were not carried forward to GRCh38/hg38 are shown.

The following color/score key is used:

item
color		item
score		type of change from hg19 to hg38
 0Contigs in hg19 dropped in the transition to hg38
 500Different portion of the same contig used in hg38 compared to hg19
 1000Updated version of the same contig used in hg38 to correct errors in the hg19 sequence

You can use the score filter to eliminate or include the different categories in the display.

Methods

The contig coordinates were extracted from the AGP files for the assemblies. Contigs that match the same name, same version, and the same specific portion of sequence are noted as identical in the two assemblies. Contigs not satisifying those identical characteristics are included in this track.
