Description

This track shows the locations of the alternate sequences/haplotypes defined by the GRC (Genome Reference Consortium) to the GRCh38 (hg38) reference sequence. The GRC generates multiple representations (haplotypes/alternate loci) for regions that are too complex to be represented by a single path.

Alt Map

The Alt Map track is the PSL alignment representation from the NCBI GFF/cigar mapping files. This PSL representation allows the side-by-side alignment to be viewed in the genome browser. This track follows the display conventions for PSL alignment tracks. Alignments are displayed in black and, depending on the track configuration settings, may be interspersed with vertical orange lines.

Haplotypes

Items in the Haplotype track mark the location on the reference genome for the corresponding sequence contained in the alternate sequence/haplotype. Data points in this track are always displayed in blue.

Click on the items in this track to use the Corresponding position: link to navigate to the alternate sequence/haplotype from the reference sequence or from the alternate sequence/haplotype to the corresponding reference sequence location.

Methods

Haplotype/Alt Map Tracks

The locations of the alternate sequence/haplotypes were established by NCBI and specified in the alignment files at:
ftp://ftp.ncbi.nih.gov/genbank/genomes/Eukaryotes/vertebrates_mammals/Homo_sapiens/GRCh38/ALT_REF_LOCI_*/alt_scaffolds/alignments/*.gff.

The GRC uses the NCBI Genomic Aligner (NG Aligner) to align the alternate loci specifically to the primary assembly chromosomes on which their anchor sequence is found.

The NCBI GFF/cigar annotation files were converted to PSL format with the kent command line program: gff3ToPsl.
