Description

This track shows N-clusters (i.e., clusters that do not have an EST that uniquely maps to the genome) determined by a large-scale study of Anopheles gambiae EST and cDNA sequences structured into the AnoEST database, which has been accepted for publication in Genome Research. The analysis summarizes 215,634 A. gambiae expressed sequences grouped into clusters using the genome sequence as template. See the AnoEST database downloads page for more information and downloads.
