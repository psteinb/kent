To attach this hub to other genome browsers not on the above list, from the blue navigation bar, go to:
  My Data -> Track Hubs
select the My Hubs tab and enter this URL:
  http://hgdownload.soe.ucsc.edu/hubs/mouseStrains/hub.txt
in the entry form, press the Add Hub button to add the hub.

This will go to the gateway page where you will find the mouse strain
hub at the top of the phylogenetic tree diagram on the left of the page.
