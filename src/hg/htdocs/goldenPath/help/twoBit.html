TwoBit Sequence Archives
 

A twoBit file is a highly efficient way to store genomic sequence.

To create a twoBit file, follow these steps:

  1. prepare the sequence for your twoBit file in a FASTA formatted file.
  2. run the faToTwoBit program on your FASTA file.
    faToTwoBit ricCom1.fa ricCom1.2bit
