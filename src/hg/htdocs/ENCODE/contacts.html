 Contact the ENCODE DCC
 

How to contact us:

  • Specific questions on how to access ENCODE data using the UCSC Genome Browser software should be directed to the public Genome mailing list; please refer to the UCSC Genome Browser Project contacts page for more information.
  • To report server issues, please refer to the UCSC Genome Browser Project contacts page for more information.
  • To obtain additional methods details for ENCODE data, or to report data problems, please contact the providing ENCODE production group, which is listed in the Contacts section of the track description.
  • To contact the ENCODE Data Coordination Center at UCSC: encode@soe.ucsc.edu.
